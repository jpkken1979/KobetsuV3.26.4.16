## Que se hizo
- **Toggle "自動反映" en shouheisha.tsx**: Switch junto al botón "工場情報を反映" que controla si los horarios se auto-rellenan desde la factory
- **Separacion de `applyFactoryDefaults`**: dividida en dos funciones:
  - `applyFactoryDefaults` —单价、业务内容、シフトパターン、勤務日数、Supervisor全部 → siempre se ejecuta al seleccionar工場
  - `applyScheduleDefaults` — 開始時刻、終了時刻、休憩 → solo cuando toggle ON
- **Nuevo componente**: `src/components/ui/switch.tsx` (shadcn)
- **Actualizacion CLAUDE.md**: se agregó documentación de `/shouheisha` y `useFactoryCascade`

## Decisiones tecnicas
- El toggle controla **solo horarios** (no todo applyFactoryDefaults) — el usuario queria que 単価 y otros campos sí se auto-rellenen siempre
- Default `autoFillEnabled = true` para backward compat (si el toggle no esta visible, elold behavior se mantiene)
- Dos useEffects separados: uno sin dependencia de `autoFillEnabled` (ejecuta siempre), otro con (controla solo horarios)
