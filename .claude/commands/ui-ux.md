Activar el sistema definitivo de diseño UI/UX (ui-ux-superskill).

Leer `.agent/skills/ui-ux-superskill/SKILL.md` y aplicar el flujo de 10 módulos:

1. **Arquetipo** — Elegir dirección estética (Editorial, Brutalist, Minimalist, etc.)
2. **Tokens** — Generar design tokens W3C (color OKLCH, spacing, typography)
3. **Theme** — ThemeProvider con dark mode (light/dark/system)
4. **Componentes** — CVA + compound components + 10-state matrix
5. **Responsive** — Mobile-first, fluid, container queries, touch targets
6. **Motion** — Animaciones con propósito, reduced motion, Framer variants
7. **CRO** — Conversión (si es landing/producto)
8. **Auditoría** — 15 dimensiones de validación visual
9. **Accesibilidad** — WCAG 2.2 AA completo
10. **Quality gate** — Checklist final antes de entregar

Usar con: `/ui-ux` o cuando el usuario pida diseño, componentes, landing, dashboard, formularios, temas, o cualquier trabajo visual.
