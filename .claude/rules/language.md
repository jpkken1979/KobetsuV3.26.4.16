# Regla: Política de Idioma — CRÍTICO

Aplica a **todas** las respuestas IA en este repositorio.

| Contexto | Idioma |
|----------|--------|
| Respuestas al usuario | **Español** (sin excepciones) |
| Documentación y comentarios | Español |
| Código (variables, funciones, clases) | **Inglés** |
| Logs técnicos | Inglés |
| Git commits y PRs | Español |

**Nunca** mezclar idiomas en el mismo bloque de código o respuesta sin justificación técnica.
