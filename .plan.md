# Plan: 一括作成 (Batch Contract Generation)

## Overview
Add a batch contract generation mode that auto-creates 個別契約書 for all employees across selected factories/lines of a company, grouped by billingRate, with a single click.

## Architecture

### New Backend Endpoint
**`POST /api/contracts/batch`** in `server/routes/contracts.ts`

Accepts:
```json
{
  "companyId": 1,
  "factoryIds": [5, 6, 7],   // optional — if empty, use ALL active factories for company
  "startDate": "2026-04-01",
  "endDate": "2026-06-30"
}
```

Logic per factory:
1. Load factory data (supervisor, job description, work hours, etc.)
2. Load active employees for that factory (`factoryId` + status=active)
3. Group employees by `billingRate || hourlyRate || factory.hourlyRate`
4. For each rate group → create one contract with:
   - All factory defaults (supervisor, complaint handlers, work conditions)
   - `contractDate` = startDate - 2 business days
   - `notificationDate` = startDate - 3 business days
   - `hourlyRate` = group rate, OT/night/holiday auto-calculated
   - `employeeAssignments` with individual billingRates
5. Cap `endDate` at factory `conflictDate` if applicable
6. Write audit log entries

Returns:
```json
{
  "created": 12,
  "contracts": [{ "id": 1, "contractNumber": "KOB-202604-0001", "factoryName": "...", "employees": 3 }],
  "skipped": [{ "factoryId": 8, "reason": "no active employees" }]
}
```

### New Frontend Route
**`/contracts/batch`** — new file `src/routes/contracts/batch.tsx`

#### UI Layout (single page, no wizard):

**Section 1: Company Selection**
- Company dropdown (same as cascading-select step 1)

**Section 2: Factory/Line Selection**
- "全部" checkbox → selects all factories/lines for that company
- Tree-style checklist: Factory → Department → Line (with employee count per line)
- Each line shows: name, hourlyRate, employee count, conflictDate

**Section 3: Contract Period**
- Start date input
- End date input
- Info: shows if any factory's conflictDate would cap the end date

**Section 4: Preview (before generation)**
- Table showing what will be generated:
  - Factory/Line | Employees | Rate Groups | Contracts to create
- Total summary: "X contracts for Y employees across Z lines"

**Section 5: Generate Button**
- "一括作成" button
- Progress bar during generation
- Results summary after completion with links to created contracts
